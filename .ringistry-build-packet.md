## 0 · One-time page setup

In Elementor, click the **gear icon** at the bottom-left of the left panel → **Page Settings**.

- **Title:** Home — Redesign Draft (so it's obvious in the list)

## 5 · Go-live checklist

When the draft looks right:

1. **Publish** the draft (set status from Draft → Published).
2. **Settings → Reading → Your homepage displays** → select "A static page" → Homepage: **Home — Redesign Draft**.
3. **Appearance → Menus** → primary menu:
   - Replace `#about` anchor with `/about/` (after you build the About page)
   - Replace `#whyus` with `/why-us/`
   - Replace `#contact` with `/contact/`
4. Footer **Useful Links** widget:
   - Fix the duplicate "Privacy Policy" — replace one with `Shipping & Returns`
   - Add: `Shop`, `About`, `FAQ`, `Contact`
   - Fix copyright to dynamic year shortcode: `Copyright © [year] Ringistry` or use Astra's `[copyright]` shortcode
5. **WP Rocket → Purge Cache** so visitors see the new homepage immediately.
6. Once live, trash the old page (12461) **OR** keep as backup labeled "Home (old)".
